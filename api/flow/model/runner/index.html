class StatefulRunner[source]#

Bases: Runner[RunnerOutputs], StatefulRunnerInit[RunnerOutputs]

A runner for stateful actors.

Stateful actors are objects constructed and configured during Runner.initialize. An execution returns a dict with objects for each output port.

Parameters

  • builder – the StatefulBuilder that created this runner.

  • kwargs – the configuration dict of this runner.
